# Agent Skill Optimization System

## Overview
A gradient-based automatic optimization system for Agent Skill prompts, implementing the GreaTer and PromptBridge methodologies.

## Modules
- **Optimizer**: Gradient-based optimization using small models (GreaTer).
- **Bridge**: Cross-model prompt transfer (PromptBridge).
- **Executor**: Skill execution and evaluation (iFlow).

## Installation
```bash
pip install -e .
```
